# TCC
Projeto desenvolvido pela equipe PrimaryDev no curso de Desenvolvimento de Sistemas do Senai Trilhas 5 (2025). A solução consiste em um sistema de gestão de estoque voltado à otimização do controle de entradas e saídas, redução de endereços fantasmas, melhoria da rastreabilidade dos produtos, organização da armazenagem e eficiência operacional.
